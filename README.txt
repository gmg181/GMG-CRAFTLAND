Craftland Map Upload - Firebase + Vercel

1. Go to https://console.firebase.google.com
2. Create a new project
3. Enable Firestore Database and Firebase Storage
4. Go to project settings > Web App > Copy config into firebase-config.js
5. Install Vercel CLI (https://vercel.com/docs/cli)
6. Run `vercel` in this folder to deploy

Enjoy uploading Craftland maps!
